How It Works

  1. Configure a provider and API key above
  2. Navigate to any webpage and click the extension icon
  3. Click Summarize This Page
  4. The page text is sent to your chosen AI provider
  5. A concise summary appears in the popup

Your API keys are stored locally in the browser and sent only to the configured endpoint. Chrome internal pages, extensions pages, and PDFs cannot be summarized.
